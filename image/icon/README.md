# ThinkWare Chatbot Icon Assets

Figma `ThinkWare_chatbot` 기준 노드에서 추출한 퍼블리싱용 아이콘입니다.

- SVG 파일은 Figma 벡터 노드에서 직접 export한 원본 기준입니다.
- 기준 노드: `1:34`, `1:128`
- CSS/HTML에서는 필요한 표시 크기를 지정하고, 아이콘 선명도 유지를 위해 SVG를 우선 사용합니다.
- Figma에서 SVG를 확인하지 못한 아이콘은 기존 PNG를 유지합니다.
- 신규 아이콘 추가 시 이 폴더에 저장하고 `user/design-system.md`의 Icon set 표에 먼저 등록합니다.
