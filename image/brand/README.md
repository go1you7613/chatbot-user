# ThinkWare Chatbot Brand Assets

사용자 챗봇 퍼블리싱에서 사용하는 브랜드 이미지입니다.

| 파일 | 표시 크기 | 실제 크기 | 용도 |
|---|---:|---:|---|
| `logo-ai-chat.png` | 52 x 14 | 53 x 14 | 1x `AI Chat` 로고 |
| `logo-ai-chat@2x.png` | 52 x 14 | 104 x 28 | 고해상도 `AI Chat` 로고 |

출처:

- Figma `ThinkWare_chatbot`
- 기준 노드: `1:37` / `1:38`

사용 규칙:

- HTML에서는 고해상도 대응을 위해 `srcset`을 사용한다.
- 로고를 임의로 늘리거나 찌그러뜨리지 않는다.
- `THINKWARE` 워드마크 이미지를 챗봇 헤더 로고로 사용하지 않는다.
- 웹브라우저 주소창 목업 이미지는 최종 퍼블리싱에 사용하지 않는다.
